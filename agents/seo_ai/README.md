# SEOAI - SEO优化专家

智能SEO分析和优化系统，提供全面的SEO诊断、关键词研究和内容优化建议。

## 🎯 功能特性

- **关键词研究**: 搜索量、难度、竞争度分析
- **内容优化**: 标题、描述、正文优化建议
- **技术SEO**: URL结构、速度、移动友好性检查
- **SEO评分**: 综合评分和问题诊断
- **报告生成**: 专业SEO分析报告

## 🚀 快速开始

### 运行示例

```bash
python seo_ai.py
```

### 基础使用

```python
from seo_ai import SEOAI

# 初始化
seo_ai = SEOAI()

# 关键词研究
keyword_data = seo_ai.research_keywords("AI Agent")

# 页面分析
analysis = seo_ai.analyze_page(
    url="https://example.com",
    content="页面内容...",
    title="页面标题",
    meta_desc="页面描述",
    target_keyword="目标关键词"
)

print(f"SEO得分: {analysis.score}")
```

## 📊 分析维度

| 维度 | 说明 | 权重 |
|------|------|------|
| 技术SEO | URL、速度、移动适配 | 25% |

## 🎯 SEO评分标准

- 🟢 **优秀 (90-100)**: 排名靠前
- 🟡 **良好 (70-89)**: 有提升空间
- 🟠 **需改进 (50-69)**: 需要优化
- 🔴 **差 (0-49)**: 急需改进

## 📝 内容优化

### 标题优化
- 长度: 50-60字符
- 包含目标关键词
- 添加品牌分隔符

### 描述优化
- 长度: 150-160字符
- 包含行动号召
- 添加数字增强吸引力

### 正文优化
- 关键词密度: 1-2%
- 至少3个H2标题
- 段落不超过300字
- 图片添加alt标签

## ⚙️ 配置说明

编辑 `config.json`:

```json
{
  "target_keyword_density": 0.015,
  "min_content_length": 300,
  "optimal_title_length": [50, 60]
}
```
