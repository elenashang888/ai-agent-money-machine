## 模板1：内容创作自动化流程

### 适用场景
自动生产社交媒体内容

### 工作流程
```
Day 1: 选题 → AI生成大纲 → 人工审核
Day 2: AI写作 → 人工润色 → 配图生成
Day 3: 多平台适配 → 定时发布 → 数据追踪
```

### 工具配置
- **选题**：Google Trends + ChatGPT
- **写作**：Claude/GPT-4
- **配图**：Midjourney
- **发布**：Buffer/Later
- **分析**：Google Analytics

### 预期产出
- 每周7篇文章
- 每日3条社交媒体帖子
- 每月1个视频脚本

## 模板7：销售线索自动化

### 适用场景
自动获取和培育销售线索

### 工作流程
```
流量获取 → 线索评分 → 自动培育 → 销售跟进
```

### 线索评分标准
- 访问 Pricing 页面：+10分
- 下载免费资源：+5分
- 打开邮件：+2分
- 点击链接：+3分
- 访问3次以上：+15分

### 培育序列
- 得分<20：教育内容
- 得分20-50：案例分享
- 得分>50：销售接触

### 工具配置
- **CRM**：HubSpot/Pipedrive
- **邮件**：Mailchimp
- **聊天**：Intercom

### 预期效果
- 线索转化率提升30%
- 销售周期缩短40%
